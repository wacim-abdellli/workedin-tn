🔄 Refresh Admin Session

After running the RLS fix SQL, you need to refresh your authentication session to get a new JWT token with the updated is_admin = true flag.

Why is this needed?
Your browser has cached an old JWT token that doesn't include the admin flag. The database now recognizes you as admin, but your session token needs to be refreshed.

Alternative Methods:

  1. Sign out and sign back in (simplest method)
  2. Run in browser console: 
    await window.supabase.auth.refreshSession();
location.reload();
  3. Clear browser cache and reload the page
