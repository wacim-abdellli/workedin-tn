# Lib Directory

This directory contains utility functions, configuration files, and third-party library initializations.

## Key Files

- **`supabase.ts`**: Supabase client initialization.
- **`i18n.ts`** / **`i18n/`**: Internationalization configuration.
- **`utils.ts`**: General helper functions (class name merging, formatting, etc.).
- **`logger.ts`**: Custom logging utility for structured logs.

## Usage

- Import Supabase client from `src/lib/supabase` for database operations.
- Use `cn` from `src/lib/utils` for merging Tailwind classes.
